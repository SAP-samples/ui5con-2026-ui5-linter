# UI5 Linter Workshop: Modernizing Legacy UI5 Applications

**Exercise Duration:** 45 Minutes
**Target Audience:** Professional UI5 developers
**Prerequisites:** Basic UI5 development experience, Node.js installed (test using `node -v`)

## Workshop Overview

In this hands-on workshop, you will learn how to use the [UI5 linter](https://github.com/UI5/linter) to identify and fix legacy patterns in UI5 applications. You'll work with a simple Todo application that contains several legacy code patterns, modernizing it step by step.
